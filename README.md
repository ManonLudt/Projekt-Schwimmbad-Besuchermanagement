Projekt-Schwimmbad
=====

Beschreibung
--------
An diesem Projekt werden werden verschiedenen Schritte der Projektplanung, Implementierung und Qualitätssicherung durchgeführt, um die Abläufe
des Besuchermanagments eines Schwimmbads darzustellen.

Ordner
--------
```
* Dokumentation
* Planungsdokumente  
* Implementierung
```
